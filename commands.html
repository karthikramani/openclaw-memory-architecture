Monitoring Commands

→ Emoji card: model, context %, tokens used, estimated cost

→ Appends token count to every reply

→ Appends full usage footer (tokens + cost)

/context list

→ Breakdown of what's inside the current context

Session Management

→ Start fresh session (cuts costs 40-60%)

→ Manual compaction before new task

/spawn [task]

→ Isolated subagent for heavy work
