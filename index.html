Storage

The Two Memory Files

Daily Log

memory/YYYY-MM-DD.md

  • Daily append-only log
  • Running context, tasks in progress
  • Read: today + yesterday at session start

MEMORY.md

Long-term memory

  • Curated long-term facts
  • Decisions, preferences, architecture
  • Only loaded in main private session

Key Insight: The model doesn't persist memory in its weights. Every session starts from scratch. FlexiBot reads memory files into the context window — that's how it "remembers". Large memory files = more input tokens = more cost.

Context Window — Your Token Bill

Understanding what drives your API costs

~2K–5K tokens · Fixed

3 Memory files
Varies · Grows over time
4 Skills
Per-skill × count
⚠ Conversation history
#1 COST DRIVER
💀 Tool outputs in history
Up to 180K chars — Silent killer

⚠️ THE BIG SURPRISE

One web scrape returning 180,000 characters gets stored in history. That single tool call gets resent with every subsequent message — dragging 50,000+ tokens forward forever. A session with 35 messages can easily grow to 2.9MB of transcript. Run /context detail regularly.
