Config Cheatsheet

Copy-paste ready configuration block

Optimized Configuration

{
  "agents": {
    "defaults": {
      "contextTokens": 80000,
      "bootstrapMaxChars": 10000,
      "bootstrapTotalMaxChars": 75000,
      "compaction": {
        "reserveTokensFloor": 20000,
        "memoryFlush": {
          "enabled": true,
          "softThresholdTokens": 4000
        }
      },
      "memorySearch": {
        "provider": "local",
        "sync": { "watch": true }
      }
    }
  }
}

Key Optimizations

  • • contextTokens: 80000 — Covers 85-90% of use cases
  • • bootstrapMaxChars: 10000 — Reduced from 20000
  • • memoryFlush enabled — Key facts survive compaction
  • • local embeddings — Free vs paid API
